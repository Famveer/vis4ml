# Visualization for Machine Learning - Graduate Course (2025.1)
{: .fs-9 }

The overall goal of this course is to provide students with the foundation needed to understand and extend the current state of the art in data visualization.
{: .fs-6 .fw-300 }

---
Uncover the power of data visualization! This course teaches you to transform data into compelling visual stories.  Learn core principles, design effective visualizations, and explore interactive techniques.  Gain practical skills using D3.js to visualize diverse data types and create impactful narratives.  Join us to turn data into insights.

By the end of the course, students will have acquired:

1.  An understanding of key visualization techniques and theories, including data models, graphical perception, and visual encoding and interaction methods.
2.  Exposure to several common data domains and corresponding analysis tasks, including multivariate data, geospatial data, and networks.
3.  Hands-on experience in creating and evaluating visualization systems.
4.  Ability to read and discuss research papers from the visualization literature.
